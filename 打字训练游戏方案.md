# Three.js + GSAP + Rapier.js 打字训练游戏开发方案

## 核心设计理念

### 1. 游戏主题：星空打字训练
- **背景**：动态星空/宇宙场景
- **玩家**：3D 字符飞船/星球
- **敌人**：陨石/外星生物（携带字母）
- **目标**：输入字母摧毁陨石

---

## 技术栈选择理由

| 技术 | 用途 | 优势 |
|------|------|------|
| **Three.js** | 3D渲染 | 丰富的3D效果，WebGPU支持 |
| **GSAP** | 动画引擎 | 流畅的文字动画、爆炸特效 |
| **Rapier.js** | 物理引擎 | 真实的碰撞检测、重力效果 |

## 核心玩法设计

### 方案A：陨石防御模式 ⭐ 推荐
```
场景：3D太空，底部有防护墙
机制：
  - 陨石从上方下落，每个携带随机字母
  - 玩家按键 → 发射激光 → 击中陨石 → 陨石爆炸（GSAP动画）
  - Rapier处理碰撞检测和物理效果
难度：
  - 速度逐渐加快
  - 陨石大小变化
  - 多字母单词模式
```

### 方案B：星球保护模式
```
场景：3D星球在中央
机制：
  - 敌人向星球移动，携带字母
  - 玩家打字 → 发射炮弹 → 摧毁敌人
  - Rapier处理敌人轨道运动
特色：
  - 敌人从不同角度攻击
  - 爆炸碎片受Rapier重力影响
```

### 方案C：文字冲浪模式
```
场景：赛博朋克风格隧道
机制：
  - 字符块向玩家高速移动
  - 玩家输入 → 字符变色 → GSAP特效
  - 连击系统触发粒子爆炸
特色：
  - 高速刺激
  - 3D隧道透视效果
```

## 推荐方案详细设计：陨石防御模式

### 1. 场景元素

```typescript
// 3D场景组成
interface GameScene {
  // 玩家
  player: {
    ship: THREE.Mesh,        // 3D飞船模型
    laser: THREE.Mesh,       // 激光束
    position: THREE.Vector3  // 底部中央
  },

  // 敌人（陨石）
  enemies: {
    mesh: THREE.Mesh,        // 陨石3D模型
    letter: THREE.Mesh,      // 3D字母（浮在陨石上）
    letterChar: string,     // 显示的字母
    velocity: THREE.Vector3  // 下落速度
  }[],

  // 物理世界
  physics: {
    world: World,            // Rapier物理世界
    bodies: RigidBody[],     // 物理刚体
    colliders: Collider[]    // 碰撞体
  }
}
```

### 2. 游戏流程

```
1. 初始化
   - Three.js场景（星空背景、光照）
   - Rapier物理世界（重力、碰撞检测）
   - GSAP时间轴（动画队列）

2. 游戏循环
   - 生成陨石（随机字母、位置、速度）
   - 玩家按键 → 发射激光
   - Rapier检测激光与陨石碰撞
   - 命中 → GSAP爆炸动画 → 分数+1
   - 未命中 → 陨石撞击地面 → 生命-1

3. 游戏结束
   - 生命归零
   - 显示分数、打字速度（WPM）
```

#### 3.2 GSAP 动画特效
```typescript
import gsap from 'gsap'

// 陨石下落动画
const spawnMeteor = (meteor: Meteor) => {
  gsap.to(meteor.mesh.position, {
    y: -10,
    duration: 3,
    ease: 'none',
    onComplete: () => removeMeteor(meteor)
  })
}

// 激光发射动画
const fireLaser = () => {
  gsap.to(laser.mesh.scale, {
    y: 50,
    duration: 0.2,
    ease: 'power2.out',
    onComplete: () => {
      gsap.to(laser.mesh.scale, { y: 0, duration: 0.1 })
    }
  })
}

// 爆炸特效
const explodeMeteor = (meteor: Meteor) => {
  // 主陨石消失
  gsap.to(meteor.mesh.scale, {
    x: 0, y: 0, z: 0,
    duration: 0.3,
    ease: 'back.in(2)'
  })

  // 粒子爆炸
  const particles = createParticles(meteor.mesh.position)
  particles.forEach(p => {
    gsap.to(p.position, {
      x: p.position.x + Math.random() * 10 - 5,
      y: p.position.y + Math.random() * 10 - 5,
      z: p.position.z + Math.random() * 10 - 5,
      duration: 1,
      ease: 'power2.out'
    })
    gsap.to(p.material, {
      opacity: 0,
      duration: 1
    })
  })
}
```

#### 3.2 Three.js 3D文字
```typescript
import { TextGeometry } from 'three/examples/jsm/geometries/TextGeometry.js'
import { FontLoader } from 'three/examples/jsm/loaders/FontLoader.js'

// 创建3D字母
const create3DLetter = (char: string) => {
  const loader = new FontLoader()
  loader.load('/fonts/helvetiker_regular.typeface.json', (font) => {
    const geometry = new TextGeometry(char, {
      font,
      size: 1,
      height: 0.2
    })
    const material = new THREE.MeshStandardMaterial({
      color: 0x00ffff,
      emissive: 0x00ffff,
      emissiveIntensity: 0.5
    })
    const mesh = new THREE.Mesh(geometry, material)
    return mesh
  })
}
```

## 总结

推荐使用**陨石防御模式**，因为：
1. ✅ 技术栈完全匹配（Three.js渲染+GSAP动画+Rapier物理）
2. ✅ 3D效果丰富（陨石、激光、爆炸粒子）
3. ✅ 可扩展性强（等级、皮肤、成就）
4. ✅ 游戏性平衡（简单易懂，有挑战性）
5. ✅ 性能可控（可优化空间大）

预计开发时间：**4-6周**
