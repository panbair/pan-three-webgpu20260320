# ⚡ 量子风暴特效

## 特效描述

量子风暴特效是一个基于 WebGPU Compute Shader 的震撼粒子系统，融合了漩涡、闪电和能量光环的视觉效果。

## 核心特性

### 🌀 中心能量漩涡
- **30万粒子**形成漩涡状运动
- 强大的向心力吸引所有粒子
- 切向旋转力形成真实的漩涡效果
- 粒子跟随漩涡路径向中心聚集

### ⚡ 闪电粒子系统
- **500个闪电粒子**间歇性爆发
- 每30帧触发一次闪电效果
- 闪电粒子随机跳跃（模拟真实闪电）
- 纯白色粒子，GPU动态闪烁效果

### 💫 能量光环
- **8个动态光环**从中心向外扩散
- 交替旋转（顺时针/逆时针）
- 脉冲式缩放动画（呼吸效果）
- 霓虹配色：青色→紫色→黄色→红色

### 🎨 动态颜色系统
- **GPU实时计算**霓虹渐变
- 基于粒子位置的动态色相
- 时间驱动的颜色流动效果
- 5种霓虹色平滑过渡

## 运镜流程

```
1. 入场动画 (2s)
   ├─ 相机推近
   └─ 能量光环淡入 + 脉冲

2. 运镜动画 (22s)
   ├─ 俯视漩涡 (4s)
   ├─ 环绕俯冲 (4s)
   ├─ 近景漩涡中心 (3s)
   ├─ 底部仰视 (4s)
   ├─ 侧面全景 (4s)
   └─ 远景 (3s) ← 触发 onComplete

3. 自动清理 (1s)
   ├─ 所有元素淡出
   └─ 执行 cleanup()
```

## 开发要点

### 关键优化
1. **双重 Compute Pipeline**：漩涡 + 闪电分离计算
2. **降频更新**：每2帧更新实例矩阵
3. **颜色间隔**：每800粒子更新一次颜色
4. **对象复用**：dummy/tempColor 避免GC

### 性能瓶颈
1. **CPU 渲染**：300K粒子的实例矩阵更新
2. **GPU 计算**：双Pipeline并发计算
3. **颜色计算**：GPU动态着色（实时）

### 进一步优化方向
1. 纯GPU渲染（手写WGSL shader）
2. 降低粒子数量（200K）
3. 增加降频间隔（3帧）
4. 简化几何体（Tetrahedron）
