# 极光织梦特效 (Aurora Dream Effect)

## 🌌 特效概述

极光织梦特效是基于原始 `ethereal-aurora.js` 特效的全面升级版本，使用 Three.js WebGPU 技术栈实现，融合了极光幕、水晶八面体、光子流的三层粒子系统，配合赛博极光配色和电影级运镜效果。

## ✨ 核心特性

### 0. 电影级运镜系统（8段运镜）

#### 运镜阶段

1. **俯视角度** (4s): (25, 30, 5) - 从上方俯视极光
2. **环绕左侧** (4.5s): (-20, 15, 20) - 环绕左侧欣赏
3. **底部仰视** (4s): (8, -8, 25) - 从底部仰视
4. **右侧环绕** (4.5s): (22, 10, -18) - 环绕右侧欣赏
5. **中心推进** (3s): (10, 8, 10) - 缓慢推进到中心
6. **螺旋上升** (4s): (15, 35, 15) - 螺旋上升到高空
7. **极速俯冲** (2.5s): (5, 5, 5) - 极速俯冲到中心
8. **缓慢拉远** (5s): (20, 20, 20) - 最后缓慢拉远

#### 运镜特点

- **入场完成**: 相机推近后自动触发运镜
- **自动清理**: 运镜完成后自动执行淡出清理
- **手动控制**: 支持 `stopCameraAnimation()` 停止运镜
- **平滑过渡**: 每段运镜使用不同缓动函数

#### 清理流程

1. 运镜完成 → 自动触发 `performCleanup()`
2. 淡出动画 (1.5s):
   - 所有材质 opacity → 0
   - 所有粒子 scale → 0.01
   - 相机拉远到 (50, 50, 50)
3. 最终清理 → 移除所有资源

### 1. 三层粒子系统

#### 第一层：水晶八面体 (8个)

- 使用 `OctahedronGeometry` 几何体
- 应用 `shuijing1.jpg` 纹理，增强视觉质感
- 环绕中心轨道运动，每个水晶独立旋转
- 呼吸动画：整体缩放范围 1.0 → 1.15
- 颜色循环：青色(0.5) → 蓝色(0.65)

#### 第二层：光子粒子 (3000个)

- 使用 `SphereGeometry` 粒子
- AdditiveBlending 混合模式实现发光效果
- 3层轨道系统，每层速度递增
- 脉冲动画：整体缩放范围 1.0 → 1.3
- 颜色循环：青色(0.55) → 蓝色(0.8) → 紫色(0.9)

#### 第三层：星尘雨 (2000个)

- 模拟重力下落的物理效果
- 循环下落机制：Y < -15 时重置到顶部
- 随机水平漂移
- 颜色：青-蓝-紫随机混合

### 2. 配色方案

赛博极光配色体系，与原版相比更震撼：

```
原版极光: 0.5 (青) → 0.6 (蓝) → 0.7 (紫)
升级版:   0.5 (青) → 0.55 (青绿) → 0.65 (蓝) → 0.8 (深蓝) → 0.9 (紫)
```

颜色特点：

- 高饱和度 (0.7-0.9)
- 高亮度 (0.6-0.8)
- 快速色彩循环 (colorCycleSpeed = 0.0003)

### 3. 动画系统

#### 入场动画 (7个GSAP tween)

1. **相机推近**: 从 (40, 35, 40) → (18, 12, 18)
2. **水晶弹入**: back.out(1.7) 弹性缩放
3. **光子扩散**: back.out(1.5) 延迟0.2s
4. **星尘淡入**: power2.out 从 Y = -50 升起
5. **水晶材质淡入**: opacity 0 → 0.9
6. **光子材质淡入**: opacity 0 → 0.8 延迟0.3s
7. **星尘材质淡入**: opacity 0 → 0.6 延迟0.5s

#### 持续动画

- **呼吸动画**: 2秒周期，sine.inOut 缓动
- **脉冲动画**: 1.5秒周期，sine.inOut 缓动，延迟0.3s
- **整体旋转**: crystal +rotation, photon -rotation
- **颜色循环**: 所有粒子随时间变色
- **水晶旋转**: 每个水晶独立三轴旋转

### 4. 技术亮点

#### 性能优化

- ✅ 使用 `InstancedMesh` 批量渲染
- ✅ 复用 `dummy` 和 `color` 临时对象
- ✅ 星尘降频更新 (每2帧更新一次)
- ✅ 限制 pixelRatio ≤ 1.0
- ✅ 禁用抗锯齿

#### WebGPU特性

- ✅ 使用 `three/webgpu` 导入
- ✅ `WebGPURenderer` 渲染器
- ✅ `alpha: true` 透明背景
- ✅ `RoomEnvironment` 环境贴图

#### GSAP规范

- ✅ 所有 tween 收集到 `allTweens` 数组
- ✅ 清理函数完整释放所有资源
- ✅ 支持立即清理和淡出清理

## 🎯 使用方式

```typescript
import { auroraDreamEffect } from '@/effect'

// 创建特效（返回控制器对象）
const controller = auroraDreamEffect(containerElement)

// 控制器接口：
interface EffectController {
  cleanup: () void              // 立即清理特效（不等待淡出）
  clearEffect: () => void       // 淡出清理特效（带1.5秒淡出动画）
  stopCameraAnimation: () => void  // 停止运镜动画
}

// 使用示例：
// 1. 立即清理（不等待）
controller.cleanup()

// 2. 淡出清理（等待1.5秒淡出动画）
controller.clearEffect()

// 3. 停止运镜（不影响特效）
controller.stopCameraAnimation()

// 向后兼容：也可以直接调用
const cleanupFn = auroraDreamEffect(containerElement)
cleanupFn() // 等同于 cleanup()
```

### 运镜动画流程

```
初始化 → 入场动画(2.5s) → 运镜动画(31.5s) → 自动淡出(1.5s) → 完成清理
```

- **总时长**: 约 35.5 秒
- **运镜段数**: 8 段
- **清理方式**: 自动触发淡出 + 手动控制

## 🎨 视觉效果描述

启动特效后，你会看到：

### 入场阶段 (0-2.5s)

1. **0-0.5s**: 相机从远处 (40, 35, 40) 快速推近到 (18, 12, 18)
2. **0-2s**: 水晶八面体从中心弹入 (back.out 弹性)
3. **0.2-2.2s**: 光子粒子向外扩散 (back.out 弹性)
4. **0.4-2.2s**: 星尘雨从下方升起 (power2.out 平滑)
5. **0-1.5s**: 所有材质淡入 (opacity 0 → 目标值)

### 运镜阶段 (2.5s-34s)

1. **俯视角度** (4s): 相机上升到 (25, 30, 5)，从上方俯视极光
2. **环绕左侧** (4.5s): 相机环绕到左侧 (-20, 15, 20)
3. **底部仰视** (4s): 相机下降到 (8, -8, 25)，从底部仰视
4. **右侧环绕** (4.5s): 相机环绕到右侧 (22, 10, -18)
5. **中心推进** (3s): 相机缓慢推进到 (10, 8, 10)
6. **螺旋上升** (4s): 相机螺旋上升到 (15, 35, 15)
7. **极速俯冲** (2.5s): 相机极速俯冲到 (5, 5, 5)
8. **缓慢拉远** (5s): 相机缓慢拉远到 (20, 20, 20)

### 淡出阶段 (34s-35.5s)

1. 所有材质淡出 (opacity → 0)
2. 所有粒子缩小 (scale → 0.01)
3. 相机拉远到 (50, 50, 50)

- **呼吸动画**: 水晶整体缩放 1.0 ↔ 1.15 (2s 周期)
- **脉冲动画**: 光子整体缩放 1.0 ↔ 1.3 (1.5s 周期)

- **整体旋转**: 水晶顺时针，光子逆时针
- **星尘雨**: 不断下落并循环

整体呈现**赛博朋克极光**的梦幻氛围，高饱和度的青-蓝-紫配色在黑色背景上绚丽发光。
