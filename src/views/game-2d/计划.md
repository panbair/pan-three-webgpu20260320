# 打字数练习游戏

## 📋 项目概述

字母随机出现在页面上，在键盘上按到对应的字母，字母就消失

## 🎯 核心特性

### 用gsap动画开发
- 有等级系统
- 有分数系统
- 每个字母底部有一张图片：https://zooow-1258443890.cos.ap-guangzhou.myqcloud.com/game1/g-v2-1.png，
- 字母3d立体的效果
- 字母随机出现在页面上，用gsap动画，动画设计有好看。
- 设计有一些字母有特别的效果，有加成分数，或者有加时间。
- 键盘按键按下时，字母就消失，有消失的特效动画。
- 有音乐播放：https://zooow-1258443890.cos.ap-guangzhou.myqcloud.com/music/1.mp3
