# 打字训练游戏 - 时间+5s 和等级提升特效更新说明

## 更新概述
为游戏中的"时间+5s"和"等级提升"事件添加了全新的全屏震撼特效，提升游戏视觉体验和成就感反馈。

---

## 新增特效详情

### 1. 时间+5s特效 (`time-bonus-effect`)

**触发条件**: 消除时间奖励字母（⏰图标）

**特效持续时间**: 2秒

**特效组成**:

#### 🕐 中央时间符号
- 巨大的"+5s"文字，绿色渐变发光
- 钟表图标（⏰）配合脉冲动画
- 文字缩放动画（scale 1.0 → 1.3 → 1.0）

#### ⭕ 旋转时钟环（4层）
- 4个同心圆环，逆时针/顺时针交替旋转
- 不同大小（100px → 340px）
- 不同速度（3s → 5s）
- 绿色半透明边框，透明度递减

#### 🌀 时光隧道（5层）
- 径向渐变的圆形层
- 从中心向外扩散
- 延迟播放（0.3s间隔）
- 营造时空穿越感

#### 💫 时间碎片（16个）
- 放射状向外飞射的绿色碎片
- 每个碎片有不同的角度
- 渐变绿色，模拟时间破碎

#### 🌊 流光粒子（8条）
- 垂直流光效果
- 绿色渐变，半透明
- 上下流动动画
- 45度间隔分布

#### ⚡ 闪电效果（6个）
- 金黄色闪电射出
- 强烈的发光效果
- 闪烁动画
- 增强视觉冲击力

---

### 2. 等级提升特效 (`level-up-effect`)

**触发条件**: 分数达到升级阈值（每100分）

**特效持续时间**: 2.5秒

**特效组成**:

#### 🎯 中央等级文字
- "LEVEL UP!"文字，金色渐变
- 等级数字（Lv.X），紫色渐变
- 文字弹跳动画
- 强烈的发光效果

#### ⭐ 星星爆炸（24个）
- ⭐emoji向四周爆发
- 360度均匀分布
- 边移动边旋转
- 缩放效果（scale 1 → 0.5）

#### 💍 光环扩散（6层）
- 金色同心圆环
- 从中心向外扩散
- 渐变透明度
- 不同延迟时间

#### 🚀 火箭升空
- 火箭emoji从底部升空
- 尾迹粒子效果（8个橙色粒子）
- 模拟火箭发射的成就感

#### 🌈 彩虹光环（12段）
- 12个彩色圆弧段
- 颜色从红到紫循环
- 旋转动画
- 形成完整的彩虹光环

#### 💰 金币雨（20个）
- 金币emoji从顶部落下
- 随机水平位置
- 旋转下落
- 发光效果

#### ☀️ 光束射线（12条）
- 金色光束从中心射出
- 脉冲缩放动画
- 半透明渐变
- 增强荣耀感

#### ✨ 粒子爆炸（30个）
- 彩色粒子爆发
- 6种颜色循环（红、橙、黄、绿、蓝、紫）
- 向外飞射
- 缩放消失

## 技术实现

### HTML结构
```vue
<!-- 时间+5s特效 -->
<div v-if="showTimeBonusEffect" class="time-bonus-effect-container">
  <div class="time-bonus-effect">
    <!-- 特效元素 -->
  </div>
</div>

<!-- 等级提升特效 -->
<div v-if="showLevelUpEffect" class="level-up-effect-container">
  <div class="level-up-effect">
    <!-- 特效元素 -->
  </div>
</div>
```

### 状态管理
```typescript
// 特效显示状态
const showTimeBonusEffect = ref(false)
const showLevelUpEffect = ref(false)

// 显示时间+5s特效
const showTimeBonusEffectFn = () => {
  if (showTimeBonusEffect.value) return
  showTimeBonusEffect.value = true
  setTimeout(() => {
    showTimeBonusEffect.value = false
  }, 2000)
}

// 显示等级提升特效
const showLevelUpEffectFn = () => {
  if (showLevelUpEffect.value) return
  showLevelUpEffect.value = true
  setTimeout(() => {
    showLevelUpEffect.value = false
  }, 2500)
}
```

### 触发时机
```typescript
// 在消除字母时触发
if (letter.isTimeBonus) {
  timeLeft.value += 5
  addActiveEffect('时间+5s', 'time-bonus', 3)
  showTimeBonusEffectFn() // 新增全屏特效
}

// 在升级时触发
if (newLevel > level.value) {
  level.value = newLevel
  addActiveEffect(`等级提升! Lv${level.value}`, 'level-up', 3)
  showLevelUpEffectFn() // 新增全屏特效
}
```

### CSS动画技术
- 使用SCSS `@for` 循环生成多个元素
- CSS自定义属性（`--rotation`）实现个性化动画
- 多层叠加创造深度感
- 渐变色、阴影、发光效果增强视觉冲击
- 延迟动画创造层次感

## 性能优化

1. **使用v-if控制显示**: 特效不显示时不渲染
2. **固定显示时长**: 2-2.5秒后自动隐藏
3. **CSS动画优先**: 使用GPU加速的CSS动画
4. **避免重复触发**: 检查状态防止重复播放
5. **合理延迟**: 错峰播放避免同时渲染过多元素

## 使用说明

### 测试特效
1. 开始游戏
2. 消除时间字母（带⏰图标）→ 触发时间+5s特效
3. 消除字母积累分数达到100分倍数 → 触发等级提升特效

### 自定义调整
如需调整特效参数，可修改以下值：
- 特效持续时间：修改setTimeout的延迟时间
- 动画速度：修改CSS中的animation-duration
- 元素数量：修改v-for的循环次数
- 颜色方案：修改CSS中的颜色值

---

## 注意事项

1. **防止重复触发**: 特效播放时会阻止重复触发
2. **层级管理**: z-index设置为1000，确保在最上层
3. **指针事件**: pointer-events: none避免遮挡游戏操作
4. **性能考虑**: 特效元素较多，低端设备可能需要减少数量
